## Notes & highlights
- **Most genre-relevant finds:** **Ghost Replay System** (4.9★, 61 reviews) — ghost
  racing is a Trackmania staple; **Custom Gauge Material** for a speedometer.
- **Most battle-tested car asset:** **Vehicle Variety Pack** (296 reviews).
- **Modular track building set:** Road Blockout Kit + Freeway Props + the Smart
  Spline generator cover blockout → spline-based tracks.
- **Full project templates** (good drive-system references, but heaviest to adopt):
  Ultimate Car Racing Project, Epic's Vehicle Game (UE4 only).
- **UE4-only (likely needs porting work on 5.8):** Vehicle Game, Driveable Car "Infrod".
- **Search-term notes:** "vehicle" beat "car"; "barrier"/"guardrail" were nearly dry
  (use the street-prop packs for barriers/cones); "road" alone was thin — modular
  kits came from "race track" / "modular environment".
- **Distribution method** (asset_pack vs complete_project) wasn't populated in the
  search API payload — inferred from titles only. Confirm on each listing page.
- **Total deduped usable free UE assets:** **27**.
